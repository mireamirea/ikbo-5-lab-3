load-загрузить дерево из файла с именем name
destroy-удалить дерево целиком
search key-поиск элемента key в дереве
add node data parent key-добавить узел со значением data,как key
add node data sibling key-добавить узел со значением data,как ровестник узла со значением key
add root data-добавить корень со значением data

show-вывести дерево на экран
show_stat-вывести статистику по дереву на экран
exit-выход из программы
